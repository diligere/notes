history
```

```
!8000

```

```
!!

Ctrl-[rR]

**piping and redirection**

```
>	stdout
<	stdin
2>	stderr
```



```
wc < users.list				stdin
echo "hi" > welcome			stdout
ls donotexist 2> error		stderr
```
